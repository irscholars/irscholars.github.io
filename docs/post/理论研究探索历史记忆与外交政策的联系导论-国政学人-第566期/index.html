探索历史记忆与外交政策的联系:导论

Exploring the link between historical memory and foreign policy:

an introduction

Lina Klymenko

Marco Siddi

编者按

本文既是一篇综述性论文，又是期刊《International Politics》第57卷特辑的导论。文章中介绍的文章均发表于本期刊物中。

文章导读

01

简介

外交政策的辩论充满了对历史事件的叙述。政治领导人经常用过去来做类比，以证明当前的外交政策决定是合理的。在美国和西欧，决策者为了动员民众支持对巴尔干和中东的军事干预，宣传了20世纪30年代绥靖希特勒的“教训”。在俄罗斯，人们把2014年后的乌克兰政府及乌克兰民族主义者与纳粹通敌者进行了类比，使得亲俄派被描绘成反法西斯战士。因此，亲俄派将克里米亚公投描述为一个选择，要么“回到俄罗斯”，要么陷入法西斯主义和混乱。在波兰和乌克兰，近来围绕民族主义的官方身份叙述已蔓延至外交政策领域，并引发了与邻国的紧张关系。东亚也出现了类似的情况，有关二战的民族主义和自我开脱的叙述严重损害了中日关系。

本文将“集体记忆”定义为一个社群所拥有的关于过去的共同记忆，是一种由政治行动者基于目前的社会和历史需要而构建的关于过去的主观形象。由于他们的话语权力（discursive power）和对媒体的接触，政治领导人特别善于塑造集体记忆，并使其适应外交政策目标的追求。对政治家和公众来说，重塑历史不再是探求“它到底是怎样的”，而是工具性地利用过去，以做出道德主张和推动政治行动。

本期特辑增进了我们对外交决策中历史运用的认识，提供了关于记忆与国际政治之间关系的理论、方法和经验方面的见解。特辑中的文分析了高度相关的案例，例如俄罗斯、乌克兰、中国、日本、美国、波兰、丹麦、英国、克罗地亚、塞尔维亚、波斯尼亚和意大利的外交政策。在现有文献的基础上，文章进一步研究了集体记忆和历史叙事在国际政治中的作用。文章探讨了各种各样的理论概念，揭示了记忆和外交政策如何以复杂的方式联系在一起，并提出了以下机制：（1）历史类比的应用；（2）历史叙事的建构；（3）记忆场所的创建；（4）对历史的边缘化和遗忘；（5）历史记忆的安全化。

02

历史类比的应用

Hans Mouritzen的文章运用了学习理论，调查了美国、英国、波兰和丹麦四个国家在2003年伊拉克战争中获取的“教训”如何影响他们对2013年叙利亚内战进行军事干预的决策。学习理论认为，吸取的教训和历史的类比可以影响决策，并使其合法化。由于在2013年时，伊拉克战争被普遍认为是一场失败的战争。莫里岑预计，这场战争的“教训”抑制或推翻了对叙利亚进行军事干预的动机。

分析证明Mouritzen的预言是正确的。关于西方可能对叙利亚发动军事打击的辩论，主要集中在2003年伊拉克战争中体现的（军事）风险上：依赖被曲解的情报、在没有联合国安理会授权的情况下发动袭击、“进入陷阱（entrapment）”（没有退出选择的长期冲突）的可能性、所谓人道主义干预的失败以及西方得不到好处（的状况）。因此，美国、英国、波兰等国的政治领导人，特别是议会，都不愿意甚至反对发动军事打击。只有丹麦领导人似乎准备好支持军事行动，如果他们的盟友(尤其是美国)选择这一选项的话。然而，丹麦的立场并没有背离学习理论的原则。Mouritzen指出，伊拉克的“教训”也出现在丹麦的辩论中，但被更有影响力、且存在已久的话语所取代。这种话语强调优先与美国结盟，以及对美国外交政策表示支持的重要性。

03

历史叙事的建构

Lina Klymenko的文章指出，叙事在塑造外交政策决定方面也发挥着重要作用。Klymenko调查了2014年后乌克兰的政策制定者如何利用历史叙述来支持他们的外交政策议程，其重点是反对俄罗斯，寻求北约和欧盟一体化。乌克兰领导人重新解读基辅罗斯、哥萨克酋长国 (Cossack Hetmanate)和苏联的历史，意在传达乌克兰历史上靠近西欧、远离俄罗斯的历史。在这些叙述中，“欧洲”和“俄罗斯”被概念化为不同的政治、文化和宗教空间。Klymenko研究了语言是如何被用来创造这种二分法的。例如，乌克兰领导人将欧洲定义为“家”和“家庭”，并将其与基督教、民主政治秩序和文化创新联系在一起。相反，他们倾向于用贬损的词语来描述俄罗斯（“莫斯科的部落”），并把这个国家与殖民主义、威权主义、落后和侵略联系在一起。

重要的是，Klymenko指出，这些叙述不仅仅是关于乌克兰的过去，事实上，他们主要关注乌克兰的未来。通过与西欧构建共同的历史，乌克兰领导人呼吁欧盟国家承认一个共同的过去，从而也承认一个共同的未来。因此，历史叙事被用来正当化乌克兰从俄罗斯的一体化中完全分离出来的策略，以及寻求与欧盟更深入的融合的希望。正如Klymenko所观察到的那样，乌克兰对欧洲和俄罗斯的看法，与上世纪90年代中东欧领导人为追求其国家加入欧盟和北约的目标而编造的说辞很相似。

在2015年后的波兰，民族主义、保守主义和民粹主义的政党法律与正义党主导政府期间特别清晰地提出了对国家受害者身份的叙述，以论证当前外交政策的错误。这是David Cadier和Kacper Szulecki文章的主题，它为民粹主义、记忆政治和外交政策之间的关系提供了深刻的见解。正如Cadier和Szulecki指出的那样，法律与正义党的领导人采用了历史叙事，倾向于把波兰或波兰民族框定为弱者或受害者，并将内部和外部的对手痛斥为“精英”，从而重新产生了构成民粹主义话语概念基础的二分法。波兰自由派政客被视为国内精英，而德国尤其被描绘成欧洲的霸主，即强加文化自由主义并损害波兰利益的欧盟级别的精英。

在波兰领导人关于受害者、英雄和犯罪者的历史叙事中，也出现了民粹主义的叙述逻辑。根据这些叙述，像波兰这样一个受害者和英雄的国家不能包括任何犯罪者。这就是所谓的大屠杀法（Holocaust law）背后的逻辑。它由法律和正义党提出，将有关波兰参与大屠杀或其他纳粹罪行的指控定为犯罪。这项法律引起了国际争议，尤其是以色列和美国政客的强烈反对，揭示了国内民粹主义的记忆政治是如何在国际舞台上蔓延和引发冲突的。

04

记忆场所的创造

在乌克兰的叙事中，它们通过对另一个国家(俄罗斯)的强烈、负面的他者化（othering）来寻求国际社会对自己外交政策目标的承认。正如Jelena Subotic的文章所揭示的那样，类似的叙事技巧也被运用到了其他背景中。Subotic表明，克罗地亚、塞尔维亚和波黑的塞族共和国（Republic of Srpska）的领导人利用第二次世界大战的记忆，强调彼此的历史罪行，并追求互不相容的外交政策主张。她特别强调，创伤记忆的物理地点如何成为外交政策建构的场所——尤其是作为过去暴力事件的叙述性主张的材料和视觉储存库，很容易被用于当前的外交政策要求。这些要求的范围可能各不相同：它们可以是要求外国归还、赔偿和道歉，甚至可以支持对领土的侵略性要求。此外，记忆的物理位置起着类推的作用：它们是过去发生的暴力事件的视觉提示，也是对未来的警告。尤其是他们对未来的意义层面，这使他们成为外交政策建设的关键因素。

Subotic通过分析最近克罗地亚和塞尔维亚关于雅塞诺瓦茨集中营（Jasenovac concentration camp）的叙述，为这些说法提供了证据。1941年至1945年，克罗地亚法西斯在那里杀害了80000多名塞族人、犹太人、罗姆人、共产主义者和其他人。在克罗地亚的政治叙事中，雅塞诺瓦茨的记忆从根本上被重新解释，第二次世界大战的暴行与克罗地亚人在1990年代所遭受的苦难（由于“塞尔维亚人的侵略”）联系起来，共产主义也失去了合法性，表明克罗地亚愿意加入欧洲的记忆空间。在塞尔维亚人的叙事中，难民营的历史被用来强调塞族人过去和现在的苦难，特别是科索沃问题。在塞族共和国，雅塞诺瓦茨还被用来声称塞族人是种族灭绝的受害者，从而转移了人们对他们在1990年代战争中作为肇事者的注意力，这一论点支持了塞族在波斯尼亚保留自主权的主张。

04

对历史的边缘化和遗忘

外交政策叙事不仅取决于一个国家对过去的记忆，也取决于被遗忘的东西。Marco Siddi的文章探讨了2014年后欧洲难民接收危机中意大利领导人外交政策叙事中的记忆与遗忘问题。Siddi认为，意大利叙述往往建立在忘记意大利的殖民历史，以及“好意大利人”观念的基础上。意大利人从未完全正视本国种族主义和罪恶的殖民历史，相反，二战后，国内主流叙事将他们描述为无法犯罪的“好人”和纳粹占领下的受害者。在这些论述中，意大利殖民主义要么被遗忘——要么被重新构建为一种短暂的经历，但却充满了积极的成就。六十年的殖民经历确实在意大利的民族认同建设中发挥了一定作用，因为它继承了种族歧视、仇外的陈词滥调。因此，“意大利性”与“白色”和“天主教”相一致。由于未能批判性地面对意大利的殖民历史，当意大利成为无数非洲和亚洲寻求庇护者的目的地时，种族主义形象、言论和实践在社会和政治叙事中幸存下来并重新出现。

Siddi的文章表明，遗忘是记忆政治和外交政策话语的核心组成部分，因为选择特定事件来构建叙事不可避免地意味着边缘化或排除其他历史事实。当有影响力的社会行动者剥夺个人叙述自己行为和记忆的权力时，这种边缘化就会发生。一直以来，随着社会对“好意大利人”观念的广泛接受，遗忘往往是由国家公众舆论的共谋造成的，这些舆论倾向于回避某些话题，采取一种“不想知道（wanting- not-to know）”的态度。意大利的殖民历史被国内公众所遗忘，这让一些意大利政治领导人更有理由声称，意大利可以在多边环境中发挥领导作用，因为他们声称，意大利没有“殖民传统”，因此不受殖民罪行的遗留影响。

05

历史记忆的安全化

Mykola Makhortykh在文章中分析了安全化理论与记忆政治之间的关系，考察了俄罗斯政府对2015年至2018年叙利亚内战的干预的叙述。Makhortykh认为，历史记忆在（有关）安全的语法中具有多重功能。通过与过去的创伤性事件相比较，它们可以被唤起来定义一个对所指对象（如国家）的存在威胁。它们也可以用来确定“不可恢复点（point of no return）”，即威胁对所指对象造成无法弥补损害的条件。值得注意的是，人们可以利用历史记忆来构建一种摆脱威胁的“出路”，尤其是让使用非常规手段应对威胁的做法合法化。

Makhortykh指出，由于其可塑性，不同的参与者可以使用历史记忆来对当前问题进行安全化、反安全化（counter- securitise）或去安全化（de- securitise）。这一点可以通过参考俄罗斯关于叙利亚内战的公开言论来说明。当俄罗斯总统普京和政府官员回忆起1990年代的车臣战争时，他们认为俄罗斯在叙利亚也面临着类似的威胁，从而为军事干预辩护。建制内的反对党雅布洛科（Yabloko） 则反驳了这一说法，称干预本身就是一种存在的威胁，因为它可能导致一个“新车臣”。此外，以纳瓦尔尼（Alexei Navalnyi）为首的反建制派则试图使这场辩论去安全化，他们主张叙利亚内战不会对俄罗斯构成威胁，并揭露了政府对历史的操纵利用。

译者评述

有关外交政策的影响因素一直是国际政治研究中的主题。从国家实力，物质结构到国内政治制度、意识形态、利益集团、领导人性格等都可以作为外交政策的自变量。本文强调了历史记忆与外交政策之间的复杂关系，包括历史记忆对外交政策决策的导向性作用，以及决策者塑造历史记忆以增强政策合法性的机制等等。

无论是国内社会还是国际社会，观念都是社会存在的重要组成部分，个人和国家都是具有社会性的行为体，其行为必然受到观念的驱动。观念为物质建构了社会意义，各种社会因素的施动性不在于它们“本来是什么”，而在于它们“被认为是什么”。客观条件固然限制了行为体选择的范围，但行为体的主观认识决定了究竟要做出哪一种选择。

在国际政治研究中，国家是基本的行为体和研究对象，然而不同于一般组织，国家内部往往具有一个多元化的社会和复杂的决策程序，究竟国内社会中的哪种观念能够代表国家这一行为体，如何测量和操作，是一个研究难点。历史记忆似乎可以成为测量国家整体观念的切入点。一方面，历史事实是已经发生过的实践，具有客观性，不以人的意志为转移，而历史记忆是主观的。历史记忆作为一个研究变量的优点是，历史事实可以作为历史记忆的客观标准测量标准。通过事实和记忆的对比可以发现国家强调了哪些历史事实，淡化或遗忘了哪些历史事实，以此测量和分析国家的价值取向。另一方面，国家作为一个社会共同体，内部群体对于当下问题和未来道路的理解和主张可能是分歧巨大的，但是对于历史的认识一定具有内部的共识或者主流的理解。历史记忆作为一个研究变量的优点是，历史记忆相对来说变化缓慢，更加稳定。因为共同体本身在历史中形成，共同体的共同意识，价值观念在历史中塑造，对历史的共同记忆维系着共同体的存在。所以，历史记忆相对于部分观念因素，可能更具有客观性和稳定性。本期的《International Politics》特辑的几篇文章都在如何操作历史记忆这一变量进行外交政策分析方面做出了示范，希望感兴趣的读者阅读。
